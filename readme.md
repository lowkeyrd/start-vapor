# Vapor 应用开发说明

Vapor 是基于 Swift 编程语言而开发的最流行的 Web 网络框架，这是一种更先进、强大且类型安全的编程语言。通过 Vapor，你可以使用 Swift 编写后端、Web 应用 API 或 HTTP 服务，与传统的服务器语言相比具有许多优点。

```shell script
s init start-vapor -d start-vapor
```

- 开始部署

```shell script
cd start-vapor && s deploy -y
```
